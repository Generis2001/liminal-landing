Liminal: The Future of Web3 Casino

1. Executive Summary

Liminal is a next-generation decentralized casino built on the Arc Testnet. It bridges the gap between traditional Web2 user experiences and the decentralized, trustless nature of Web3. By leveraging advanced wallet abstraction and gasless infrastructure, Liminal provides a cinematic, frictionless, and highly secure gaming environment.

2. Core Features

  • Frictionless Onboarding: Powered by Privy. Users can connect effortlessly via social logins (Google, Twitter), email, or any standard Web3 wallet (MetaMask, WalletConnect). Instant account creation removes the barrier to entry for non-crypto natives.
  • Built on Arc Testnet: Experience zero-latency transactions and non-existent gas fees. The Arc infrastructure guarantees an uninterrupted flow, allowing gameplay to occur entirely on-chain without the traditional UX friction of constant wallet approvals.
  • Cinematic Gameplay: An immersive dark-mode UI featuring fluid micro-interactions, glassmorphism aesthetics, and real-time state updates.
  • Absolute Security: Every bet is written to smart contracts. Users retain absolute custody of their assets with transparent, decentralized logic that is provably fair.

3. Technology Stack

The Liminal platform is divided into two main components: the core Casino Web Application and the standalone Marketing Landing Page.

Frontend Architecture

  • Framework: Next.js 15 (App Router)
  • Language: TypeScript
  • Styling: Tailwind CSS v4, Vanilla CSS
  • Animations: Framer Motion (for dynamic UI layouts and micro-interactions)
  • Icons: Lucide React

Web3 & Authentication Infrastructure

  • Authentication: Privy (Wallet abstraction, Social Logins, embedded wallets)
  • Blockchain Interaction: Wagmi & Viem (for contract reads/writes and listening to chain state)
  • Target Network: Arc Testnet (Custom RPC integration)

4. UI/UX Design System

Liminal utilizes a highly customized, premium dark-mode design system.

  • Typography: Outfit for display headings (bold, geometric, attention-grabbing) and Inter for highly legible body text.
  • Color Palette: Deep dark backgrounds (#0a0a0b) contrasted with sleek, metallic gold accents (#c4a97a and #e8d5b0) to emulate a premium, high-roller environment.
  • Glassmorphism: Heavy use of backdrop-blur (e.g., backdrop-filter: blur(20px)) combined with semi-transparent borders to create depth and modern spatial interfaces.

5. Network Configuration (Arc Testnet)

To interact with Liminal, users connect to the Arc Testnet using the following parameters:

  • Network Name: Arc Testnet
  • RPC URL: https://testnet.arcscan.app/rpc
  • Explorer: https://testnet.arcscan.app/

6. Project Repositories & Deployment

  • Liminal Casino App: The core Web3 application featuring the full Privy connection flows, smart contract interactions, and the casino dashboard. Deployed on Vercel.
  • Liminal Landing Page: A completely isolated, standalone marketing repository designed specifically for presentation and SEO optimization. It maintains the identical visual identity of the core app but strips out the heavy Web3 authentication dependencies to ensure maximum loading speeds for marketing purposes.
